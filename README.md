# NumberTode
translate written numbers into numerals

## Examples
```
NumberTode("three") //3
NumberTode("forty-two") //42
NumberTode("NINETY NINE") //99
NumberTode("two hundred") //200
NumberTode("three hundred and twenty two") //322
NumberTode("One Hundred, Twenty-Three") //123
NumberTode("two thousand and twelve") //2012
NumberTode("One-Thousand, Three-Hundred and Seven") //1337
NumberTode("three point one four") //3.14
NumberTode("zero point five") //0.5
NumberTode("three plus two") //3 + 2
NumberTode("two hundred and one add seven minus four") //201 + 7 - 4
NumberTode("twelve divided by three") //12/3
NumberTode("a hundred shared into ten equal groups") //100/10
NumberTode("one over four") //1/4
NumberTode("One hundred and two plus three over four") //102 + 3/4
NumberTode("(One hundred and two plus three) over four") //(102 + 3)/4
```
